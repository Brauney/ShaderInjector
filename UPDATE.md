# Update Guide

For existing users who already have the mod installed and want to learn how to update.

Updating is simple, simply drag the contents of the new zip file to the game directory again and replace any changes *(You will not lose your shader replacements)*.

When you boot into the game the shaders need to be recompiled in order to use the new changes so just open the Shader Injector menu, and for your shader replacements click each one and click Rebuild Replacement Shader.

Done!
