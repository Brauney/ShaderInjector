# Shader Injector
Shader Injector Mod for FF7 Rebirth PC. This is a DX12 *(D3D12)* based mod that intercepts the rendering API calls, and allows you to inject/replace the original shaders within the title with modified ones.

# Installation Guide

How to install the mod!

I cannot stress enough that you must follow the guide down to the letter as close as possible if you want to setup the mod quickly without much pain. Fortunately it's not too difficult to setup
