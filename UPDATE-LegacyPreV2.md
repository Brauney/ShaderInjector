# Update Guide *(Pre v2.0)*

For existing users who already have the mod installed and want to learn how to update.

**NOTE: You do not need to rebuild the game shader cache, or any of that. Changes should apply and remain persistent.**

Updating is simple, simply drag the contents of the new zip file to the game directory again and replace files *(You will not lose your shader replacements)*.

<p float="left">
    <img src="GithubContent/Update/step1.png" width="49%" />
    <img src="GithubContent/Update/step2.png" width="49%" />
</p>

When you boot into the game the shader replacements you setup will need to be recompiled in order to use the new changes so just open the Shader Injector menu, and for your shader replacements click each one and click Rebuild Replacement Shader.
